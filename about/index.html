关于本站

了解 TGFind 的创立初衷、核心特色、网站流量公开公示以及如何支持我们。

TGFind 起名自 Telegram Navigation ，是一个致力于收录与整理 Telegram（电报）上优质公开频道、群组和机器人的导航平台。我们从海量数据中进行人工筛选与过滤，旨在为广大电报用户提供一个干净、清爽、高效的入口，帮助大家发现优质内容。

我们划分了三大核心板块：

频道导航

精心收录各领域的优质推送频道，获取最新行业资讯、资源与趣闻。

群组导航

收录活跃度高、探讨氛围良好的公共群组，助您找到同好交流分享。

机器人

搜集各种功能强大、好玩实用的电报小助手，扩展您的电报生态体验。

核心特色

  • 严格的防垃圾与清爽体验

    我们有专人对收录的条目进行全方位核验，屏蔽虚假流量、黑灰产广告、以及恶意钓鱼群组，打造纯净的电报生态导航。

  • 自研智能收录系统

    配合我们自主研发的 Telegram 收录机器人 @TGQRYbot，可实现极速提报、分类调整、元数据自动更新。

  • 开源与透明

    我们始终相信开源的力量。TGFind 的前端代码全部在 GitHub 开源，同时，我们也对网站的日常流量进行完全公开的透明公示。
